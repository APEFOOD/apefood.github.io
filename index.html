FRAMEWORKS



Laravel 5

The creator(s) of the Laravel framework believe that happy developers write great code. Laravel is an incredibly polished framework with many conveniences like the aptly named Eloquent ORM, PHPUnit integration, & many more. With Laravel, the resultant web application should be just as pleasing to management & end users as the process of coding it usually is to developers.

Symfony 3.x

You would be hard-pressed to find a web application written in PHP that does not use at least one Symfony component. This framework is made up of tools that let you develop and build even the most complicated web applications in a very comprehensible manner. A lot of, if not all, PHP frameworks rely on Symfony for some of their core functionality. There is nothing you can't do with Symfony.

WordPress

WordPress is probably the world's most popular content management system (CMS). The main reasons behind that are it's flexibility and simple user interface. In all honesty, the WordPress code base is a nightmare to work with. But in instances where usability takes precedence over everything else, WordPress could be your best bet. Many people are familiar with it, and if you have the patience for it you can make the software bend at your will.

JavaScript/jQuery

I like to think of JavaScript as the magic of the web. It lets you supercharge your HTML with interactivity, animations, and visual effects. The JavaScript jQuery library makes building interactive web pages a fast and seemingly effortless task. A true blessing for web designers & developers all around the world.
